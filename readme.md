## 🚀 Deploying to GitHub Pages

1. Create a new repo on GitHub called `ai-study-plan`
1. Upload `index.html` to the repo
1. Go to **Settings → Pages**
1. Set source to **Deploy from a branch → main → / (root)**
1. Your site will be live at `https://YOUR_USERNAME.github.io/ai-study-plan`
